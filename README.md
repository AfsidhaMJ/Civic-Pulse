# Community Problem Solver

A hackathon-ready civic tech platform to report and manage local issues.

## Tech Stack
- HTML, CSS, JavaScript
- Python Flask
- Local JSON database

## How to Run
1. Go to backend folder
2. Run: `python app.py`
3. Open frontend/index.html in browser

## Hackathon Pitch
This platform empowers citizens to report local issues and ensures transparent, authority-driven resolution without complexity.
